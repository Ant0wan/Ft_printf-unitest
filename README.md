---

## Installation

```shell=
git clone https://github.com/Ant0wan/Ft_printf-unitest.git && cd Ft_printf-unitest/
```

Copy your ft_printf soucre code in the repository:

```shell=
cp -f [Makefile+auteur+*.c+*.h+...] Ft_printf-unitest/rendu/
```
